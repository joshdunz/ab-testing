A/B Testing Appointment Booking Website

Excel/CSS | March 2024

1. About

The task of the project was to conduct A/B Test on a given webpage and another webpage with suggested improvements :

  • I sought to prove that my editted page with higher contrast increased accuracy and cut down time spent on the booking site
  • The testing helped me determine whether the adjustment of my target levers were effective at improving the user experience
  • Broadly the levers I chose were:
    • Using primary colors
    • Using Higher contrast font colours
    • Using font weights to demarcate categories of information

2. Initial website problems

  • The website has a low contrast color scheme making text hard to read
  • The buttons on the right: "See Appointment" and "Schedule Appointment" are not distinguisable
  • The date section and details section are not clearly demarcated

3. Changes

  • Dark blue was used to demarcate the buttons using contrast
  • The date section color was changed to blue to show clear separation from details
  • The Details section font color was changed to black from grey for greater contrast

4. A/B Testing Metrics and Calculations

4.1 Misclick Rate

Null Hypothesis: The misclick rate of the original website is the same as the updated website

  • Prediction & Rationale: Reject as original website has poor contrast making users click the first seemlingly correction option which based on the appointment type

Alternative Hypothesis: The misclick rate of the original website is different than that of the updated website

  • Prediction & Rationale: The contrast is high for name descriptions enabling users to focus on accuracy as opposed to deciphering

Test chosen: Chi-squared test

  • Rationale: The the outcome I was testing for was categorical (True/False)

Significance: We find a statistically significant difference in the rate of misclicks for version A and version B, as p<=0.05(0.00056). Since the number of times a misclick was recorded on Version A was 9/16 which is higher than Version B (0/15), we see evidence that less users misclick on version B than A.

description of key values:
  • df: there is only 1 logically independent variable which is a boolean of whether a misclick occurred
  • chi^2: 11 in the context of this test with a p of 0.00056 shows a significant difference between observed and expected values

Conclude: We reject the null hypothesis

4.2 Time spent on page

Null Hypothesis: The average time spent the original page is lower than the updated website

  • Prediction & Rationale: Reject as users spend more time trying to read text than select and option

Alternative Hypothesis: The average time spent the original page is higher than that of the updated website

  • Prediction & Rationale: Users can clearly understand text making it easier to select an option

Test chosen: two t-test

  • Rationale: Data was continous and we were testing whether there was a significant difference in average time spent on the website in both directions, not just one.

Significance: We find a statistically significant difference in the rate of misclicks for version A and version B, as p<=0.05(0.00035). Since the average time recorded on Version A (29411ms) is higher than Version B (8173ms), we see evidence that users spend less time on version B than A.

description of key values:
  • t-score: negative t-score means that Sample B average is lower than Sample A
  • chi^2: 11 in the context of this test with a p of 0.00056 shows a significant difference between observed and expected values

Conclude: We reject the null hypothesis and find statistically significant evidence that the alternative hypothesis is true that the average time on page for Version A is higher than Version B

4.3 Time till the first click

Null Hypothesis: The average time to first click of the original page is lower than the updated website

  • Prediction & Rationale: Reject as users spend more time trying to understand what information the text provides and what the organization of it is

Alternative Hypothesis: The average time to first click of the original page is higher than that of the updated website

  • Prediction & Rationale: Users can clearly make out how the text hierarchy signals information and make decision faster

Test chosen: two t-test

  • Rationale: Data was continous and we were testing whether there was a significant difference in average time to first click in both directions, not just one.

Significance: We find a statistically significant difference in the rate of misclicks for version A and version B, as p<=0.05(0.00166). Since the average time to first click recorded on Version A (10452ms) is higher than Version B (5234ms), we see evidence that users spend less time till their first click on version B than A.

description of key values:
  • t-score: negative t-score means that Sample B average is lower than Sample A

Conclude: We reject the null hypothesis and find statistically significant evidence that the alternative hypothesis is true that the average time to first click for Version A is higher than Version B
